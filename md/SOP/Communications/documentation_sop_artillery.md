# Использование артиллерии

Артиллерийская поддержка во время миссий может быть обеспечена двумя способами:

- по запросу к выделенному арт. расчету из игроков
- по запросу к Game Server Officer

### Фактическая процедура оказания поддержки

Акронимы и понятия:

- *FM* - Fire mission, огневая задача для артиллерии;
- *SUP Net* - Support Network, частота ДВ для операций поддержки (ДВ 51)
- *Бекрид* - чтение полученного запроса отправителю с целью подтверждения данных;
- *TRP* - Target/terrain reference point (ти ар пи), отмеченные на карте характерные позиции (элементы рельефа, сооружения, путепроводы);
- *ETA* - Estimated time of arrival (и ти эй), примерное время прибытия/подлета;
- *FFE* - Fire for effect, артиллерийский огонь согласно запросу - все запрошенные выстрелы;
