FireTeam Формации

Формации для FireTeam (команды)

Шеврон

Wedge - Шеврон (левый и правый)
Основное построение FireTeam, универсальное построение с максимальным обзором, которое позволяет команде быстро выстроиться в линию в любом направлении.
Вы всегда по default'y ходите в шевроне. ВСЕГДА. Это основное построение.
Плюсы: хороший обзор, возможность быстро перестроиться в линию в любом направлении
Минусы: небольшая безопасная зона стрельбы по ходу движения

Линия

Line - Линия (левая и правая)
Основное боевое построение, заметив врага или на подходе к вражеской позиции команда должна выстроиться в линию
в направлении противника с расстоянием 10-15 метров друг от друга для достижения максимальной огневой эффективности и выживаемости (огневая мощь распределена равномерно, нет скученности, граната не убьет всех сразу)
Плюсы: высокая плотность огоня, низкий шанс ранить союзника,
Минусы: фланги остаются открытыми, сложно сместить огонь на фланги

Файл и Битая колонна

File - Файл Staggered Column - Битая колонна
Используется для передвижения в условиях затрудненной видимости, например в лесах или сильном тумане.
FTL может меняться местом с AR.
Плюсы: сложно потеряться, быстрый переход в линию при фланговой атаке
Минусы: перестроение в линию вперед и назад занимает много времени 		Построение движения используемое для перемещения по дорогам, в составе с техникой, существует в левой и правой вариации
Плюсы: фланги закрыты, быстрое перестроение в линию при атаке с обоих флангов
Минусы: низкая плотность огня в направлениях вперед и назад

Squad Формации

Формации для Squad (отряда)

Отрядная Колонна

Squad Column - Отрядная Колонна ( Шевроны )
Основное построение для движения в составе отряда
Плюсы: большой обзор, быстрое перестроение в линию в любом направлении
Минусы: малая зона безопасного огня вперед и назад.

Отрядная Линия

Squad Line - Линия ( Шевроны )
Основное построение для движения в приблизительном направлении противника
Плюсы: быстрое перестроение в линию, хорошая защита флангов
Минусы: малая зона безопасного огня вперед и назад

Боевая Линия

Squad Line - Линия ( Боевая )
Основное боевое построение отряда, используется в бою с противником
Плюсы: высокая плотность огня, трудно попасть в союзника
Минусы: открытые фланги, сложно сместить огонь на фланги

Отрядный Файл и Битая Колонна

Squad File - Отряд файлом Sq. Stg. Column - Отряд битой колонной
Построение для передвижения в условиях затрудненной видимости, например в лесах или сильном тумане
Плюсы: сложно потеряться, быстрый переход в линию при фланговой атаке
Минусы: перестроение в линию вперед и назад занимает много времени 		Построение для передвижения по дорогам в сопровождении техники
Плюсы: фланги закрыты, быстрое перестроение в линию при атаке с обоих флангов
Минусы: низкая плотность огня в направлениях вперед и назад
